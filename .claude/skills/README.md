# Skills Index

Browsable index of skills in `.claude/skills/`. Grouped by category (the prefix before the first `-` in each skill's directory name). This file is for humans — Claude Code discovers skills by scanning `SKILL.md` files directly, regardless of this index.

New skills should be added here by [dev-new-skill](dev-new-skill/SKILL.md) when created.

## dev

- dev-grill-me — relentless interview to sharpen a plan or design, one question at a time
- dev-conversation-handoff — generate a concise handoff summary to continue work in a new chat
- dev-feature-summary — generate long-term technical documentation for a completed feature
- dev-new-skill — interview-driven creator for new skills in this project's convention

## docs

- docs-app-map — investigate the codebase and write topic-organized CLAUDE.md files per module
- docs-sync-on-commit — update the relevant topic CLAUDE.md(s) before each commit
